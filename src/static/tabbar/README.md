# TabBar Icons

This directory contains the tabbar icons for the 4-tab navigation.

## Required Icons

The following icon files are referenced in `pages.json` and need to be created:

1. **home.png** / **home-active.png** - 首页 tab
2. **pet.png** / **pet-active.png** - 档案 tab  
3. **fun.png** / **fun-active.png** - 玩法 tab
4. **user.png** / **user-active.png** - 我的 tab

## Specifications

- Size: 81px × 81px (recommended for WeChat Mini Program)
- Format: PNG with transparency
- Inactive state: Gray (#999999)
- Active state: Primary brand color (#FF8A65)
